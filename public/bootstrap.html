Hello, Twitter Bootstrap!

The point of this livecoding exercise is:

  • To demonstrate how Bootstrap works.
    • Create nicely styled components without having to write much (if any) CSS and JavaScript!
    • HTML (data-)attributes everywhere!
    • Accessibility attributes
  • To demonstrate that Bootstrap's online documentation is often wrong or confusing.
    • ...and how to work around those issues using the Chrome DevTools inspector.
  • To provide accurate sample code for some popular Bootstrap components.
